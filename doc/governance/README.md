# 治理流程文档

此目录包含项目治理相关的文档。

## 文档类型

- 治理合规性报告
- 变更管理流程
- 风险管控机制
- 质量保证体系

## 治理原则

所有治理文档必须遵循项目宪法的治理与修订原则：
- 宪法作为最高约束力
- AI Agent 必须遵守宪法约束
- 修订流程必须规范化
- 定期进行合规性检查
