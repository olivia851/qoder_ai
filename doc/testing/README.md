# 测试文档

此目录包含系统测试相关的文档。

## 文档类型

- 测试策略和计划
- 单元测试规范
- 集成测试方案
- 性能测试标准
- 自动化测试框架
- 测试环境管理

## 质量要求

所有测试文档必须遵循项目宪法的质量内建原则：
- 测试覆盖率 ≥80%
- 核心业务逻辑必须可测试
- 关键用户旅程需要E2E测试覆盖
- 性能基线必须满足要求
